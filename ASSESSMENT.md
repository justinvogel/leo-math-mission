# How we figure out where Leo is (without a “grade 3” label)

## The rule

**Never tell Leo he’s at a grade level.**  
Skills aren’t a single rung. A kid can be solid on multiplication facts and shaky on unlike denominators in the same week. Grade labels flatten that and hurt motivation.

What we track instead:

| Status | Meaning |
|--------|---------|
| **Solid** | Usually gets it (~80%+) — maintain, don’t drill forever |
| **Building** | Sometimes gets it — good training zone |
| **Growth edge** | Often misses — train here next (this is the catch-up lever) |
| **Not scanned** | No data yet |

Parent view may show *“often introduced ~grade 4–5”* as **curriculum context for you only**. Leo sees skill names and status, not grades.

---

## Layer 1 — In-app Skill Scan (tonight)

Built into Math Mission:

1. Leo runs **Skill Scan** (~12–18 min, ~20 questions).
2. Covers foundations → fractions → decimals → light stretch.
3. Produces a **training path** (top growth edges).
4. Nightly missions auto-bias toward that path.

Re-run from Parent view anytime (after a few weeks of practice is ideal).

This is **formative** (guide practice), not a formal psych-ed evaluation.

---

## Layer 2 — Free outside diagnostics (this week / this month)

Use these as a **second opinion**, not a replacement for daily practice.

### A. Tutoring-center free assessment (best external map)

Mathnasium, Sylvan, Huntington often run **free placement / learning assessments**.

- You get a skill breakdown by topic  
- Zero cost for the assessment  
- Pitch: “We’re building a catch-up plan; we want a map.”  
- You do **not** have to buy a package  

**Do this once** so you’re not guessing from the bleachers.

### B. Khan Academy (daily practice companion — no API)

Khan’s public API was **removed** years ago. We **cannot** auto-import his mastery into Math Mission.

What works well manually:

1. Create a parent-linked Khan account for Leo.  
2. Start him in **Get ready for 6th grade** or **5th grade math**, not randomly.  
3. Use **Course challenge** / unit quizzes when offered — screenshots of unit mastery are enough.  
4. Mirror growth edges into Math Mission’s training path (or re-run Skill Scan).  

Khan + Math Mission: Khan for videos/explanations; Mission for short gamey evening reps.

### C. School MAP / baseline (when enrolled)

Whichever school he lands in — ask for **early MAP (or equivalent)** scores and the **domain breakdown** (Numbers & Operations, etc.). That’s the “official” snapshot. Pair it with our skill map; they should roughly agree.

### D. Full psych-ed eval (only if needed)

~$1,500–3,000, only if free diagnostics + school data still look like something deeper (ADHD, dyslexia, processing). Not step one.

## Layer 3 — Other apps (honest take)

| Tool | Role | Integrate with Mission? |
|------|------|-------------------------|
| **Khan Academy** | Free curriculum + videos | Manual only (no API) |
| **Khanmigo** | AI tutor (paid) | Use alongside, don’t sync |
| **IXL** | Huge skill tree + diagnostic | Paid; manual skill list |
| **ALEKS** | Adaptive placement | School-ish; overkill for home start |
| **Prodigy / other games** | Engagement | Fine for fun; weak as sole diagnostic |
| **Mathnasium app / center** | Assessment + tutoring | Best external skill map |

**Bottom line:** True “plug in Khan and auto-place Leo” isn’t available to a home app. The robust home pattern is:

**Skill Scan (Mission) → daily missions → optional free center diagnostic → school MAP when available.**

---

## What “caught up for 6th” means here

Not one number. Ready enough when these are mostly **Solid**:

**Foundations:** mult/div facts, place value, multi-digit mult/div  
**Fractions:** parts, equivalent, compare, +/− same denom, unlike denom, multiply (div intro)  
**Decimals:** place value, compare, ↔ fractions, +/−, multiply  
**Stretch:** story problems, percent basics  

Until then: train **growth edges**, not “all of math.”
